Code Next Connect

The place to connect Code Next students, mentors, coaches and Googlers.

Why is it so great?

Code Next Connect (CNC) was made for Code Next students that would like more opportunities to meet each other. We know that it hasn’t always been the easiest to meet other members and professionals, so this site will fix that. You are able to view the other members of Code Next’s profiles which include their interests, hobbies, skills, favorite projects they’ve done. They can also contact Code Next coaches and other computer science professionals for help and mentoring, growing their network.
